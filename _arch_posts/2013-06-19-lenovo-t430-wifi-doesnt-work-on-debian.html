Well, all because the firmware's are distributed as a non-free package.

1. add non-free to your apt-get (edit /etc/apt/sources.list)
2. do apt-get update
3. install 'firmware-iwlwifi' (sudo apt-get install firmware-iwlwifi)
4. reboot
